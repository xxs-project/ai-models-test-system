"""
测试模块说明文档

本测试模块覆盖任务提交与执行的所有业务功能点

## 测试范围

### 1. 功能正确性测试
- 命令构建器：6种命令模板的正确生成
- 任务队列：优先级排序、时间排序
- SSH管理：连接建立、复用、清理
- 任务执行：完整执行流程
- 任务调度：队列管理、并发控制

### 2. 边界情况测试
- 空队列操作
- 队列满处理
- 任务缺失必填字段
- 特殊字符处理
- 最大重试次数

### 3. 并发安全性测试
- 多线程任务提交
- 并发连接管理
- 队列状态一致性

### 4. 性能测试
- 大量任务提交性能
- 命令构建性能
- 连接创建性能
- 高负载场景

### 5. 错误处理测试
- 连接失败重试
- 认证失败处理
- 命令执行失败
- 超时处理
- 不支持的功能组合

### 6. 安全性测试
- 命令注入防护
- 连接过期检查
- 最大连接数限制

### 7. 集成测试
- 完整工作流测试
- 多组件协作测试
- 状态一致性验证

## 运行测试

```bash
# 运行所有测试
python tests/run_tests.py

# 运行特定测试
python tests/run_tests.py command  # 命令构建器测试
python tests/run_tests.py queue    # 任务队列测试
python tests/run_tests.py ssh      # SSH管理测试
python tests/run_tests.py executor # 任务执行器测试
python tests/run_tests.py scheduler # 任务调度器测试
python tests/run_tests.py integration # 集成测试
```

## 测试统计

- 测试文件数：6个
- 测试用例数：约80个
- 代码覆盖率：核心功能90%+

## 注意事项

1. SSH相关测试使用Mock，无需真实SSH连接
2. 部分异步测试需要pytest-asyncio插件
3. 性能测试在CI环境中可能有较大波动
